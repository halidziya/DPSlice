Dirichlet Process Slice Sampler
Executable file obtained with Visual Studio 2015. The project depends on FastMat library.
Matlab and Python wrappers give example about how to use the program.
Main program is dpsl.exe first parameter is data matrix rest is optional according to order : 
dpsl.exe datafile.matrix [hypermean.matrix] [hyperscatter.matrix] [params.matrix (d,m,kappa,gamma)]  [#ITERATION] [#BURNIN] [#SAMPLE]  [initiallabels.matrix]
